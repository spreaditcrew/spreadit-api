## Description

Application which is created on [Nest](https://github.com/nestjs/nest) framework for SpreadIT cms api purpose.

## Requirements

  - docker
  - npm

## Installation

Copy file `.dist.env`, create `.env` and set variables 

```bash
$ npm install

$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
